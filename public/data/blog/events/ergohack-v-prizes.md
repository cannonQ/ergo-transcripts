# ErgoHack V Prizes


> **Published:** 2022-09-18 | **Author:** Ergo Platform | **Tag:** Events | **Type:** blog

**Related concepts:** ERGOHACK, Ergo Foundation, Registers, SigmaUSD

---

With Ethereum’s merge, we are witnessing a seismic shift in the hashrate landscape for Proof of Work blockchains. In the last few days for instance, Ergo’s network hashrate has jumped to over 100TH/s from approximately 30Th/s.

The Ergo Platform is seeing impressive growth and a lot of the credit is due to the Ergo Foundation’s commitment to hosting regular hackathons. If you are not already aware, the next ErgoHack is scheduled for October 3-20, 2022. Applications are currently being accepted until October 3rd and this ErgoHack will promote the theme of mining and minting.

This iteration of ErgoHack features an impressive list of partners! We are proud to co-present this hackathon with the Djed Alliance, Flux, and Alephium. Teams that register for ErgoHack V will be in the running for the largest prize pool to date for Ergo hackathons as a result of these partnerships. Prizes for the Ergo Foundation, Flux, and Alephium will be decided based on the following criteria:

- Innovation
- Code quality
- Completed implementation
- Quality of user interface

### Ergo Foundation Prizes

**1st Place: 10k SigUSD
2nd Place: 6k SigUSD
3rd Place: 2k SigUSD**

### Flux Prizes

Flux uses Docker Hub containers to distribute applications in a decentralized, and censorship-resistant manner. A Docker container image is a lightweight, standalone, executable package of software that includes everything needed to run an application: code, runtime, system tools, system libraries and settings.

Due to Docker being code agnostic, Flux can run any kind of software. Front- or Back-end deployments are common on the Cloud. For ErgoHack V, the Flux judges will be looking at how participants explore use cases for this technology.

**1st Prize: $7k FLUX
2nd Prize: $3k FLUX
3rd Prize: $1k FLUX**

### Alephium Prizes

To be considered for the Alephium prizes, competitors are encouraged to build multi-chain applications or services (ie. multi-chain wallet, data service, dual-miners, etc) that include Alephium as part of the build.

Alephium shares a common ground with Ergo in terms of ethos and technology. For ErgoHack V, Alephium is encouraging collaborations between like-minded projects and communities for the purposes of building multi-chain products and tools. The Alephium prize pool will consist of:

**1st Prize: 10k ALPH
2nd Prize: 7k ALPH
3rd Prize: 3k ALPH**

### Djed Alliance Prizes

The Djed Alliance will provide prizes of 1800 DACAU (Djed Alliance Contribution Accounting Unit) to any of the submissions that are judged to bring significant contributions to the work of Djed/SigUSD.

Have an idea for ErgoHack V? Be sure to get your application in by October 2nd! Head to the [ErgoHack V website](https://ergohack.io/) for all the details.
