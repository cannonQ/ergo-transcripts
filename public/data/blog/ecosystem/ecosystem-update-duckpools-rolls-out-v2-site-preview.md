# Ecosystem Update: Duckpools Rolls Out V2 Site Preview

> **Published:** 2025-11-26 | **Author:** Ergo Platform | **Tag:** Ecosystem | **Type:** blog

**Related concepts:** Collateral, Duckpools, Native tokens, Slippage, Transaction

---

The team behind **Duckpools**, a prominent lending and borrowing protocol on the Ergo blockchain, has released a comprehensive development update. With a focus on improved user experience, smarter contract logic, and transparency, the team has successfully deployed a preview of their V2 interface alongside a detailed roadmap for the coming weeks.

## The V2 Site Preview is Live

The biggest news for the community is the launch of the updated test site. Users can now visit [test.duckpools.io](https://test.duckpools.io/) to experience the latest UI changes and dashboard upgrades.

**Key Features of the Update**:
- **Smarter Proxy Contracts**: A major backend improvement removes the need for slippage, which previously caused some transactions to fail on the v1 site.
- **Enhanced Dashboards**: Users can now manage and view upgraded statistics and information on existing v1 pools and positions.
- **Historical Data**: Comprehensive pool dashboards now include history, and a dedicated lender dashboard has been completed.

**Note for Users**: While the test site is live, the team notes that the database is currently undergoing a resync (expected to complete within 24 hours). Additionally, as some calls are currently unoptimized, users may experience slower load times in specific areas like the borrow dashboard. Speed optimizations are scheduled for deployment after the full V2 pool launch.

#### Community Feedback Needed:
_The team encourages the Ergo community to test the new dashboards and report any statistics that appear incorrect. While basic manual testing has been completed, duckpools.io remains the recommended entry point for major transactions until the beta phase concludes._

## Recent Completed Milestones

Leading up to this release, the Duckpools developers have delivered a significant list of technical improvements, including:
- **Continuous Yield Delivery** deployed on token pools.
- **Smart Reserves**: Support for Automated Exit Thresholds for borrowers.
- **Pricing Logic**: Deployed quote contracts with advanced collateral pricing logic.
- **Maintenance**: Reduced interest update fees, resolved v1 rate delay bugs, and completed an internal contract audit.

## The Road Ahead

Duckpools has outlined a transparent roadmap for the remainder of the year and beyond.

**Target: December 8th**

The next sprint focuses on expanding the Smart Reserves functionality and documentation:
- **UI Implementation**: Smart Reserves UI for v2 pools.
- **Documentation & Templates**: Release of templates and docs for Smart Reserves, Quote contracts, and Interest Rate contracts.
- **Testing**: Release of v2 token test pools for public feedback.

## Future Outlook

Following the December updates, the team is looking toward several major ecosystem goals, including:
- Publishing a full **Security Audit**.
- releasing v2 ERG test pools.
- **Open-sourcing** the frontend code.
- Enabling decentralized hosting support.
- Optimizing high-volume off-chain bots (including mnemonic support).

We look forward to seeing the community test the new interface and provide feedback to help shape the future of lending on Ergo!
