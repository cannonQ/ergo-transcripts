# Ledger Launches Ergo Support in Developer Mode


> **Published:** 2023-01-10 | **Author:** Ergo Platform | **Tag:** Tooling | **Type:** news

*For a while now, the Ergo community has been excitedly awaiting Ledger support for the Ergo token. The Ergo Foundation is pleased to announce some exciting news for Ledger device owners: Ergo is now available for download on Ledger Live via “Developer Mode.”*

**Related concepts:** Ergo Foundation, Native tokens

---

What does this mean? To be clear, this is not the final product that will be released on Ledger. However, it does allow Ergonauts who are interested in accessing Dev Mode features the ability to access Ledger’s software support for Ergo tokens. For those who would like to start using Ledger’s Ergo app in Dev Mode, please complete the following steps:

1. Open the Ledger Live Application on your personal computer.
2. On the left side of the Ledger Live Application, select the “My Ledger” tab.

![photo_2023-01-12 17.46.51.jpeg](https://storage.googleapis.com/ergo-cms-media/photo_2023_01_12_17_46_51_884a31feda/photo_2023_01_12_17_46_51_884a31feda.jpeg)

3. Unlock your Ledger device via your PIN if you have not already done so.
4. In the top right corner of the Ledger Application, select the “Settings” tab (toothed wheel icon).
5. In Settings, select the “Experimental features” tab.
6. Slide the “Developer mode” toggle on.

![photo_2023-01-12 17.46.48.jpeg](https://storage.googleapis.com/ergo-cms-media/photo_2023_01_12_17_46_48_07ae8008d6/photo_2023_01_12_17_46_48_07ae8008d6.jpeg)

7. Select the “My Ledger” tab that you accessed in Step 2 (above).
8. Search for Ergo in the “App catalog” (this search field is available at the top of the listed tokens).
9. Select Ergo and install the app on your Ledger hardware device.

![photo_2023-01-12 17.46.45.jpeg](https://storage.googleapis.com/ergo-cms-media/photo_2023_01_12_17_46_45_b079ef153e/photo_2023_01_12_17_46_45_b079ef153e.jpeg)

If you choose to use this application, please be aware that the Ledger developer team provides the following warning for all experimental features:

>“These are experimental features we provide on an “as is” basis for our tech savvy community to test. They may change, break, or be removed at any time. By enabling them, you agree to use them at your own risk.”

Although this is only in Developer mode, it is very good news for those who have been hoping to use their Ledger devices with Ergo tokens. It is hoped that the official Ergo app will be available to Ledger users very soon. 

If you hold Ergo tokens and do not want to use Developer mode on Ledger (but would like the comfort of cold storage immediately), you still have the option of using either the [Ergo Paper Wallet](https://ergopaperwallet.org/) or [Ergo Mobile Wallet](https://ergoplatform.org/en/ergo-wallet-app/) to securely store your assets.

Stay tuned to Ergo’s social media channels for future updates on Ledger’s Ergo support!
