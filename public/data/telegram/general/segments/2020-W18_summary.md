# Ergo General Chat — 2020-W18 Summary
**Period:** 2020-04-27 to 2020-05-03 | **Messages:** 230 | **Participants:** 21

---

## Key Topics Discussed

- **Mining Pool Launch & GPU Support**: Multiple mining pools in development (jasondavies pool, ergopool.io). HiveOS support confirmed for AMD miners; foundation offering collaterals to first testers. Autolykos algorithm confirmed as ASIC and pool-resistant PoW.

- **Token Issuance & Explorer Updates**: ~130 tokens already issued on-chain by users, mostly experimental. Explorer planned to show issued tokens and unconfirmed transactions.

- **Mixer & Privacy Features**: Non-interactive mixer PoC in development using ErgoScript with clever register-based variable handling. Explorer API integration allows mixing in light wallets and CLI scripts. Token mixing and stablecoin mixing identified as high-impact but controversial use cases.

- **Cross-Blockchain Atomic Swaps**: External product announcement pending; details uncertain but online presence of both parties not required.

- **Light Node Service Economics**: Ergo design allows users to run light/full nodes and earn fees via privacy/decentralization-promoting services without mining hardware.

- **Stablecoin & DeFi Readiness**: Team uncertain if Ergo is ready to be backbone for new economic models via real DeFi. Stablecoin experiments continuing.

- **Community Growth & Exchange Listings**: Ergo listed on ProBit, HotBit, Tidex, Citex, Waves.exchange, GJ.com, ViteX, P2PB2B. One major exchange integration in progress; next target tier-1. Community strength emphasized as prerequisite for top-tier listings (e.g., Binance).

- **Distributed Threshold Signatures**: Real threshold signatures (distinct from multisig) in development.
