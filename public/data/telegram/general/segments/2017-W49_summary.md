# Ergo General Chat — 2017-W49 Summary

## Key Topics Discussed

- **Testnet Debugging Round**: Community-wide testing initiative launched with bounty rewards for bug reports. Full nodes required initially; light nodes to follow. Unique node naming required for bounty qualification.

- **Improved Authenticated Dynamic Dictionaries (IADD)**: Core Ergo foundation protocol under testing. Waves-NG protocol implemented IADD on 2017-12-05, yielding favorable results for reducing blockchain bloating. Expected to inform Ergo's implementation approach.

- **Token Distribution & EFYT**: 50 billion tokens planned for first year emission across potential dual-chain structure (MoneyChain and DataChain). EFYT (Ergo First Year Tokens) tradeable on Waves DEX with BTC or WAVES; will convert to mainnet ERG upon launch with possible holder discounts.

- **Mining via Mobile Devices**: Planned feature enabled by IADD protocol; expected post-mainnet launch (target: 2018). GPU mining confirmed as immediate option.

- **Waves Collaboration**: Ergo and Waves acknowledged as independent projects sharing common founders/colleagues, collaborating on NG protocol testing and authenticated dictionary implementation while maintaining separate roadmaps.

- **Proof of Work Consensus**: Ergo confirmed as PoW chain; sidechaining support planned to enable cross-chain events (e.g., MoneyChain/DataChain coordination).
