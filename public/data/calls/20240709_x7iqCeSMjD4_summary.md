# Ergo Blockchain Community Call Summary
**Date:** July 9, 2024  
**Call #:** 124th Dev Update

## Executive Summary

This dev update featured a special guest appearance by Mr. Pixel, creator of the popular Ergo-based game Cyberverse. The call provided deep insights into Cyberverse's development journey, from its humble beginnings as 250 hand-drawn 24x24 pixel NFTs called "Ergo Pixels" to its current state as a comprehensive multiplayer gaming ecosystem. Mr. Pixel discussed the game's evolution, cross-chain expansion plans, and upcoming survival/open-world features while addressing community feedback on game mechanics and crafting systems.

The technical updates covered significant progress across multiple Ergo ecosystem projects, including Nautilus Wallet version 0.11.0 release, Sigma State Interpreter development for Protocol 6.0, Spectrum Pool's mobile proxy contract deployment, and Rosen Bridge's continued cross-chain infrastructure development. The call demonstrated the active development happening across the Ergo ecosystem, with Cyberverse serving as a flagship example of what can be built on Ergo's platform.

## Key Discussion Points

- **Cyberverse Origins**: Started as hand-drawn 24x24 pixel NFTs, evolved into comprehensive gaming platform
- **Cross-chain Integration**: Expanding to Cardano and other blockchains while maintaining Ergo as primary chain
- **Game Development**: New multiplayer racing game with 20-player capacity, AI bot integration, and tiered car system
- **Future Roadmap**: Survival/open-world gameplay with bigger maps, mob combat, and resource gathering
- **Community Feedback**: Active integration of player suggestions for crafting recipe rebalancing
- **Technical Updates**: Nautilus Wallet improvements, Sigma State Interpreter progress, Rosen Bridge scaling tests
- **Economic Model**: Free-to-play with NFT ownership benefits, internal marketplace with significant trading volume

## Decisions Made

- Cyberverse will remain primarily on Ergo while expanding cross-chain functionality
- Crafting recipes will be rebalanced based on community feedback to reduce energy/stamina costs
- Racing game implements tiered system to ensure fair competition between different car classes
- CYPX token will utilize Rosen Bridge for cross-chain functionality
- Cyber Citizens NFTs will remain Ergo-exclusive with potential centralized bridging options
