# Ergo Ecosystem Weekly Update - Sep 22nd, 2024

### Post #1 — @kushti (2024-09-22)

* Nautilus wallet v0.14.0 with support for Transaction History released!
* Ergo Foundation has approved a $12,000 grant for CyberVerse, an innovative gaming project on the Ergo blockchain. This grant will fund marketing initiatives, community engagement, and ecosystem expansion, driving more users to CyberVerse and Ergo alike!
* Ergo reference protocol client 6.0.0-alpha1 for devnets released! [Release Ergo protocol reference client 6.0.0 Alpha1 (for devnets only!) · ergoplatform/ergo · GitHub](https://github.com/ergoplatform/ergo/releases/tag/v6.0.0-alpha1)
* Mew Finance released v1 whitepaper [x.com](https://x.com/Mew_finance/status/1835727821123014968?t=KxqflO8YKOul0VN-2cOVvQ)
* Lithos developer Cheese Enthusiast published a forum post on Decentralized Mining and Non-Interactive Share Proofs [Decentralized Mining and Non-Interactive Share Proofs](https://www.ergoforum.org/t/decentralized-mining-and-non-interactive-share-proofs/4860) . The project is looking into different NISP contracts using 6.0 devnet at the moment
* Developer of [Sigmaspace.io](http://Sigmaspace.io) is working on explorer api which is compatible to ergoplatform explorer api.  
  Implemented 3 endpoints so far and will implement more in the coming days. First endpoints and documentation for api can be found here: [Ergo Explorer Api Reference](https://api.sigmaspace.io/docs#/)
* Fleet SDK 0.7.0 with a bunch of new features and improvements released!
* Bruno is looking for devs for new degen game based on hodlcoin pools. Please reach him or kushti if interested.

Do not forget to play with and support new applications launched this Summer, such as orderbook based decentralized exchange TradeHouse [Trade House](https://ergoauctions.org/trade) , and Gold pegged ERG-backed algorithmic stablecoin Gluon <https://www.gluon.gold/>

And do not forget about next emission reduction coming in 14 days! [ErgoWatch](https://ergo.watch/dashboards/emission)
