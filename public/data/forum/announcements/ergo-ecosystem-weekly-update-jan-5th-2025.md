### Post #1 — @kushti (2025-01-06)

Here is the post-holidays digest covering Ergo ecosystem news and updates since Dec, 16th, 2024.

* Another emission reduction happened on Jan, 5th! Now the network is rewarding its miners with constant reward of 18 ERG per block, which is about 12,960 ERG per day on average.
* first draft of finalized “sub-blocks” (input/ordering blocks) EIP published [Input/Ordering Block Implementation by kushti · Pull Request #2055 · ergoplatform/ergo · GitHub](https://github.com/ergoplatform/ergo/pull/2055/files#diff-3b625a79f97550190afbfef5c3c2aec8d1b5e83d6b74ab1b2b4fcf1844293072)
* Ergo protocol reference client 5.1.1 released [Release Ergo Protocol Reference Client 5.1.1 · ergoplatform/ergo · GitHub](https://github.com/ergoplatform/ergo/releases/tag/v5.1.1)
* Paideia went through a bugfix in contracts successfully and back!
* Support for silver datafeeds added to oracles! in [Silver oracles by Alesfatalis · Pull Request #329 · ergoplatform/oracle-core · GitHub](https://github.com/ergoplatform/oracle-core/pull/329)
* Work towards Nautilus mobile started
* Cornell Ergo payment system is done
* Keystone integration: SDK part is mostly finished and pr is opened in draft [[WIP] Ergo support by Alesfatalis · Pull Request #105 · KeystoneHQ/keystone-sdk-rust · GitHub](https://github.com/KeystoneHQ/keystone-sdk-rust/pull/105)
* Contracts design and unit tests are done for Raffle v2
* Lithos finishing up last few NISP contracts
