### Post #1 — @kushti (2024-12-02)

* A proposal for 6.0 soft-fork published! You can find shorter version of it in this forum post: [6.0 Soft-Fork Proposal](https://www.ergoforum.org/t/6-0-soft-fork-proposal/5005/1) , full version is available in EIP-50 [[WIP] EIP-50 - 6.0 Soft-Fork by kushti · Pull Request #100 · ergoplatform/eips · GitHub](https://github.com/ergoplatform/eips/pull/100)
* Ledger integration code passed functional testing on Ledger side, next, security audit!
* Rosen watcher operators, we kindly request you to update your watcher-service to version 4.0.4] ([Release Publish 4.0.4 · rosen-bridge/watcher · GitHub](https://github.com/rosen-bridge/watcher/releases/tag/4.0.4)) and ui to version 3.0.0 ([Release Publish watcher-app-3.0.0 · rosen-bridge/ui · GitHub](https://github.com/rosen-bridge/ui/releases/tag/watcher-app-3.0.0)).
* Rosen bridge now supports a lot of new Ergo tokens! Please check LP possibilities on other chains and provide liquidity!
* Mew Finance is continuing to develop community liquidity bootstrapping initiatives, please check <https://fund.mewfinance.com/>
* A new 3D visualization tool showcasing Ergo’s blockchain data, enabling users to explore the UTXO model and blockchain dynamics interactively : <https://www.3rgo.xyz/>
* Spectrum SDK update proposed by Dexy team finalized and now under review by Spectrum team: [Add native to Dexy amm contracts by code-for-uss · Pull Request #45 · spectrum-finance/ergo-dex-sdk-js · GitHub](https://github.com/spectrum-finance/ergo-dex-sdk-js/pull/45)
* Developers, do not miss new forum post from Luivatra on how to use contract templates (EIP-5) [Ergoscript and contract templates - why & how (EIP-5)](https://www.ergoforum.org/t/ergoscript-and-contract-templates-why-how-eip-5/5006)
* Ergonaut Airdrop, a space-themed command-line tool for distributing tokens to Ergo miners. Designed for community token airdrops with style [GitHub - The-Last-Byte-Bar/ergonaut-airdrop](https://github.com/The-Last-Byte-Bar/ergonaut-airdrop)
* Lithos Lithos finished up a large portion of contracts which create fraud proofs on NISPs (e.g to prove incorrect PoW, invalid header, not following NISP rules, etc.)
* Fleet SDK 0.8.0 released!
* Sigmanauts have set BlueSky accounts up! Sigmanauts bsky: [@sigmanauts.com on Bluesky](https://bsky.app/profile/sigmanauts.com) Ergo Platform bsky: [@ergoplatform.org on Bluesky](https://bsky.app/profile/ergoplatform.org)
