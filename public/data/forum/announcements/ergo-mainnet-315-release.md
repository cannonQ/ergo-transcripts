### Post #1 — @Martin-mx (2019-12-19)

This minor release includes:

* PR [#996](https://github.com/ergoplatform/ergo/pull/996): testMnemonic from config now checked in /wallet/init and /wallet/restore
* PR [#999](https://github.com/ergoplatform/ergo/pull/999): wallet password encoding specified explicitly now
* PR [#1000](https://github.com/ergoplatform/ergo/pull/1000): a failure in /wallet/restore (e.g. due to illegal key size) is caught now
* PR [#992](https://github.com/ergoplatform/ergo/pull/992): node UI (/panel) updated
* PR [#985](https://github.com/ergoplatform/ergo/pull/985): cross-build for Scala 2.11, stricter control for dependencies (no sbt-lock anymore)
