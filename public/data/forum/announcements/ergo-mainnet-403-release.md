### Post #1 — @Martin-mx (2021-02-01)

Version 4.0.3 of Ergo protocol reference client. It includes:

* [#1248](https://github.com/ergoplatform/ergo/pull/1248) /node/shutdown method to shut the node down (/node/shutdown)
* [#1246](https://github.com/ergoplatform/ergo/pull/1246) API method to collect the boxes for external applications (/wallet/boxes/collect)
* [#991](https://github.com/ergoplatform/ergo/issues/991) API methods for fee estimation (/transactions/poolHistogram, /transactions/getFee, /transactions/waitTime)
* [#1263](https://github.com/ergoplatform/ergo/pull/1263) API method to get scan-related transactions (/wallet/transactionsByScanId)
* [#1259](https://github.com/ergoplatform/ergo/pull/1259) Node panel updated

Also, mining rewards can be recognized and be added to the wallet now with wallet being locked.
