### Post #1 — @kushti (2024-09-15)

* Ergo reference protocol client 5.0.23 is finalized and under review, release is expected in few days. Please help with testing [Candidate for 5.0.23 release by kushti · Pull Request #2169 · ergoplatform/ergo · GitHub](https://github.com/ergoplatform/ergo/pull/2169)
* “Ergo: After Central Banks” ([Ergo: After Central Banks](https://www.ergoforum.org/t/ergo-after-central-banks/4852)) article is proposing to position Ergo ecosystem as a toolset for people not served well with current efforts of (central) banks as well as for people living where central banks already failed (and every central bank is failing sometimes, you know)
* Another devnet release for 6.0 protocol client is coming, now with support for unsigned big integers, higher-order functions, options and headers serialization etc. After that, there will be another big devnet release and then finalization of 6.0 soft-fork protocol update proposal.
* Cornell AAP students DAO appeared on Paidea! ( [Paideia | DAO Toolkit](https://app.paideia.im/cornellaapdao24-25) )
* Storage rent dashboard @ [sigmaspace.io](http://sigmaspace.io) : <https://sigmaspace.io/en/storage-rent> it lets you track upcoming storage rent boxes and how much rent is going to be claimed. Also overview of storage rent in the next 24h / 7 days / 30 days
* Exciting News: Storage Rent Now Live on Sigmanauts Mining Pool! Join the pool at: URL: 65.108.57.232 Low Hash Port: 3052 (For miners under 10GH/s), High Hash Port: 3053 (For miners over 10GH/s), Temporary User Interface: <http://sharkpool.ergominers.com> . Start maximizing your mining efficiency today with our new storage rent feature!
* The Stable Order is starting an apprenticeship programme ([Apprenticeship | Stability Nexus](https://docs.stability.nexus/about-us/the-stable-order/join-the-stable-order/apprenticeship)). The idea is to give the apprentices simple but interesting ideas (like hodlCoin, for instance) and help them to learn how to implement them. There is need for help of more experienced Ergo devs, who would be willing to mentor the apprentices. If anyone here would like to help and become a mentor, please reach The Stable Order.
* ChainCash got support for notes redemption and API description you can find @ [GitHub - ChainCashLabs/chaincash-rs: ChainCash Server: Free Banking for Everyone!](https://github.com/ChainCashLabs/chaincash-rs) . There is no UI at the moment , but money hackers can already do experiments using API!
* Satergo wallet ( <https://satergo.com> ) is getting a lot of new features and improvements these days, as well as Ledger support. Upcoming release release won’t include ledger though because a dev wants to make a new wallet format version before that
* UI for bridging ETH with Rosen leaked: [x.com](https://x.com/Arc11x/status/1833162664505221533)
