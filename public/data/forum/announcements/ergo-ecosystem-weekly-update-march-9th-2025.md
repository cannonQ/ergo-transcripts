### Post #1 — @kushti (2025-03-11)

This update is covering two weeks again, since Feb, 23rd! We hope to get back to weekly updates from the next update.

* There was good Community Townhall on “Ergo as the Endgame for Crypto”, you can find recording at [Ergo: Ergo as the Endgame for Crypto: Catch today's town hall on t...](https://nostr.band/note1aelp589ud3vem3nfu2whdjeqzn479ya3dsfccca59d479f7hyt0qtg3fjl) . Attend the next Townhall on “Programmable Reserve Assets on Ergo: the Quest for Better Money” <https://x.com/ergo_platform/status/1896410896827637977>
* SigUSD bank’s update box has been refreshed, so the bank is now safe in the presence of storage-rent for decades to come!
* Mew Finance now has UI for the SigUSD bank, its preview is available at <https://dev.mewfinance.com/>
* Mew also has dashboard for Duckpools loans [Mew Trading](https://trading.mewfinance.com/duckpools)
* Ergo-MCP to make AI aware of Ergo blockchain data released [GitHub - marctheshark3/ergo-mcp](https://github.com/marctheshark3/ergo-mcp) . Check how to use it : <https://x.com/i/broadcasts/1nAKEgNLrBVJL>
* SkyHarbor V2 preview is out : <https://v2ui.skyharbor.io/>
* First refactored Nautilus Walet is released <https://github.com/nautls/nautilus-wallet/releases/tag/v1.0.0-beta.1and> available on Abyss Channel <https://chromewebstore.google.com/detail/nautilus-abyss/moaohjebjfiodfiegjnjkbnalepemcpf?authuser=0&hl=en>
* Sigmanauts Mining Pool finalized Sigma Bytes front end - allow miners to mint NFT for 3 ergs which goes to the pool wallet, miner is also able to remint and burn old sigma byte for free just tx fee. Still it is the only pool which is delivering demurrage profits to miners, and demurrage profits are increasing as could be seen at SigmaSpace explorer [Sigmaspace | Storage Rent](https://sigmaspace.io/en/storage-rent)
* Ergo Node Android is getting updates again! [GitHub - rustinmyeye/ErgoNodeAndroid: A fork of NeoTerm. An attempt at a one click Ergo node app with install scripts to setup and run an Ergo node on Android.](https://github.com/rustinmyeye/ErgoNodeAndroid)
* Charles Hoskinson proposed to fork DogeCoin, with using Ergo tech <https://x.com/IOHK_Charles/status/1896791646962090427>
* reportedly, Satergo update is close
