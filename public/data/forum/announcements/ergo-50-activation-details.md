### Post #1 — @kushti (2022-11-08)

On block #824,320 ( [Ergo Explorer](https://explorer.ergoplatform.com/en/blocks/cf237375bc13610ea0e6804e12bf2d7bf85fcf3c047f92dac909e755fcd3ea42) ) voting for 5.0 was started. The voting got concluded with block #857,087, with enough (90+%) support for 5.0 given.

With block #857,088 pre-activation delay started. And 5.0 will be activated on block #889,856.

5.0 is soft-fork, so old nodes do not need to update. However, mining nodes do need for update. For other nodes, it is better to update with time also, to avoid possible invalid transactions propagation issues.
