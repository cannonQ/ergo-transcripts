### Post #1 — @Martin-mx (2020-12-16)

Candidate for 3.3.7 release. It includes:

```ergoscript
#1197 : Distributed signatures implementation
#1232 : Fix for #1229: /scan/addBox is not updating wallet balance (when scanId = 10)
#1238 : walletInteraction flag
#1239 : Unconfirmed transactions added to /scan/unspent
#1243 : Tests cases for all address types added.
#1241 & #1245 : [PANEL] Added custom fee and assets
#1244 : scorex-util, scrypto, sigma-interpreter and scorex dependencies updated
```
