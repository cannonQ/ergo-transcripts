### Post #1 — @kushti (2020-11-17)

A new collectible token made by Ergo Foundation is linked to the Ergonaut and contains certain information:

* A digital monument to our anons. Token id was built from a box which contains (in its registers) the string “anon\_92048 | anon2020\* | anon\_real” (see first output of <https://explorer.ergoplatform.com/en/transactions/495fabc3c3e695acba7fffe4b1991e28d1ab2bb7f1d39c0cd642457dc8b8a89d>) . Anons made a lot of contributions to the Ergo ecosystem: ErgoMixer, ZK Treasury, Auctions House.
* 32 bits of work (token id: 000000002306564c7b03a78b76a2ce1d9d4e748ddf462cb73cc30b48e43e6e44 )
* Ergonaut character image associated with NFT (no rights to the concept). The token issuing box contains image hash (059c321fad7e0b2bb474ebea46c8978d1957eb1b5f11c589e52f129d14b3bde4) in the register R8 (see transaction <https://explorer.ergoplatform.com/en/transactions/ca9cd54c78b26718da953373a86597b70ece79f3e5fbb8ced579a0577b1992ce>).

All the funds from this sale will go to our anon devs, enabling them to continue building cool new features for the Ergo ecosystem!

If you don’t run your own node, you can still bid on this auction using the transaction Assembler service. Click on the ‘Configure Wallet’ button in the top right corner of the Auction site, click ‘Any Wallet’, and fill in your address!

The auction will last for about 2 months, hopefully, more wallets will have tokens support by then! Initial bid is 100 Erg, min step is 100 erg!

[ergoauctions.org](http://ergoauctions.org/#/auction/active)
