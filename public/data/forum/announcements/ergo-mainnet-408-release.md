### Post #1 — @Martin-mx (2021-03-15)

Ergo Protocol Reference Client 4.0.8. It includes:

* [#1290](https://github.com/ergoplatform/ergo/pull/1290) - P2P layer optimizations (to reduce number of memory allocations)
* [#1301](https://github.com/ergoplatform/ergo/pull/1301) - synchronized added to methods modifying history database
* [#1291](https://github.com/ergoplatform/ergo/pull/1291) - V2 block checkpointing and skipping V1 transactions validations (checking only UTXO set correctness)
* [#1276](https://github.com/ergoplatform/ergo/pull/1276) - Extended key refactoring: it is made F-bounded to avoid .getInstanceOf
* [#1303](https://github.com/ergoplatform/ergo/pull/1303) - more deterministic and faster tests with using eventually
