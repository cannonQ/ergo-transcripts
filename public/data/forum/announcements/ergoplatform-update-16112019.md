# "Update from 15.11.2019

## EFYTs Buyback

Currently, all the sell orders fulfilled, even more, at the moment of writing this post there’s a buy order of 16+K. So the Foundation is swapping the tokens successfully!

## Events:

I’m going to this event in Istanbul tomorrow: “EuroAsia Cryptocurrency Investment Summit” <https://www.eventbrite.com/e/euroasia-cryptocurrency-investment-summit-tickets-74541833631> . If you’re around let’s get in touch!

In December we’re going to China again, Proof-of-Value conference (I’ll be on a panel there) and meetup @ Zhejiang university at least. More tour events to be announced.

## Ecosystem Decentralization & Anti-fragility:

A repository is started to discuss community and development practices, please help us with that!  
<https://github.com/ergoplatform/community/issue> and [Pull requests · ergoplatform/community · GitHub](https://github.com/ergoplatform/community/pulls)

## Core Development:

3.1.4 will be out next week, 3.2.0 in progress.

We have many new bounties for writing manuals etc please help us [GitHub · Where software is built](https://github.com/ergoplatform/ergo/issues)

## App Development:

Appkit library to develop application with different languages is released: [GitHub - ergoplatform/ergo-appkit: Appkit: A Library for Java/Scala/Kotlin Development of Ergo Applications](https://github.com/aslesarenko/ergo-appkit) . Soon it will be more documentation and examples!

Core has support for decentralized pools now, and the development of the first pool is making good progress (will be released before NY hopefully).

Contract examples for bonds based on Ergo [Bonds based on Ergo (or the "Yield protocol") - Ecosystem Development - Ergo Community Forum](https://www.ergoforum.org/t/bonds-based-on-ergo-or-the-yield-protocol/128) (similar to the Yield protocol <http://research.paradigm.xyz/Yield.pdf> )

There’s a team working on LETS implementation.

In the next update it will be more details on new wallets coming!"

> Update from Alex Chepurnoy
