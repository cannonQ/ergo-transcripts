Ergo Protocol Reference Client 4.0.5. Changes are:

* [#1281](https://github.com/ergoplatform/ergo/pull/1281) : description for wallet/boxes/collect added
* [#1289](https://github.com/ergoplatform/ergo/pull/1289) : the node is re-announcing new blocks received from other peers
* [#1288](https://github.com/ergoplatform/ergo/pull/1288): API method to get Merkle proof for a transaction added
* more log output for wallet failures
* fix for [#1294](https://github.com/ergoplatform/ergo/issues/1294)
