### Post #1 — @kushti (2024-11-18)

* There was good community call on Fri (Nov, 15th)! And it was decided to have more!
* Nautilus Technical Documentation page published along with dApp Connector guide. Check it out: <https://docs.nautiluswallet.com>
* Ledger UI improvements requested by Ledger HQ are in progress ( [UI improvements by steel97 · Pull Request #26 · ergoplatform/ledger-app-ergo · GitHub](https://github.com/ergoplatform/ledger-app-ergo/pull/26) )
* ergo-graphql v0.5.3 released
* Sigma 6.0.0-RC1 is finalized and going to be released!
* Paideia mainnet quietly launched! Can be found on the usual url app.paideia.im, the old test daos can be found at beta.paideia.im for now.
* ErgoPad team has decided to shut the service down after 3 years of work.
* Mining Swaps Payments are LIVE @ Sigs Mining Pool [Sigmanaut Mining Pool DAO - Sigmanauts](https://sigmanauts.com/mining)
* The Field is doing private testing of contracts and UI . “The Field” is a peer-to-pool pledging protocol where individuals can pledge assets (coins or tokens) to support a participant in a competition or market.
* ErgoHack IX roundup published [Reddit - Dive into anything](https://www.reddit.com/r/ergonauts/comments/1grugaw/ergohack_ix_roundup/)
