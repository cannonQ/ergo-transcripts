Release 3.2.7. It contains:

* Generation of block candidate with certain transactions included ( [#929](https://github.com/ergoplatform/ergo/issues/929) )
* Fix for [#1035](https://github.com/ergoplatform/ergo/issues/1035) (incorrect cost computation on signer side)
* Double-checks for inputs and outputs of generated transaction.
