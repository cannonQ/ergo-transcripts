### Post #1 — @kushti (2024-09-29)

* Ergo Foundation announced ErgoHack IX, a UI/UX oriented hackathon! ![:hammer_and_wrench:](https://emoji.discourse-cdn.com/twitter/hammer_and_wrench.png?v=12 ":hammer_and_wrench:") This is your chance to help shape the future of Ergo by designing or improving UI/UX for a smoother, more user-friendly experience. Date: October 25-31, 2024 , Location: Ergo’s Discord. Don’t miss out - REGISTER NOW @ <https://ergohack.io/register/> and check the full schedule here: [ErgoHack IX](https://ergohack.io/)
* Ergonauts are discussing ideas ErgoHack IX over developers chat ([Telegram: Contact @ErgoDevelopers](https://t.me/ErgoDevelopers) or #development channel in Discord) . About dozen of ideas already published!
* Mew Finance Mart now has “phygital” section where you can buy NFTs tied with physical goods (Ergo merchandise atm). Currently, you can find stickers there, more merch will be available in days! [Mew Mart](https://mart.mewfinance.com/explore?category=Phygital)
* Rosen team started semi-public testing of Ethereum bridge, public traces of it can be found in Rosen Telegram group [Telegram: Contact @rosenbridge\_erg](https://t.me/rosenbridge_erg) . Also, support for the watcher-service versions prior to 3.0.0 officially ended this week. To avoid any disruptions, please update your watcher-service to the latest version as soon as possible.
* Sigmanauts mining pool is beta testing swapping mining reward. Miners are able to mint miner id tokens or burn them. Mint is live at: [dev.ergominers.com](http://dev.ergominers.com)
* [Sigmaspace.io](http://Sigmaspace.io) explorer added more endpoints to its API : [Ergo Explorer Api Reference](https://api.sigmaspace.io/docs#/) , next up are endpoints used by rosen bridge watcher so sigmaspace api can be used in watcher configuration
* Dexy team got first audit results from 4Eyes and working towards addressing the audit.
* ergo-graphql 0.5.2 and Nautilus wallet 0.14.1 released
* Ergo Indonesia instagram started! [Ergo Platform Indonesia 🇮🇩 (@ergoplatformindonesia) • Instagram photos and videos](https://www.instagram.com/ergoplatformindonesia?igsh=bHZ5OGcyZzFmdTRt)
