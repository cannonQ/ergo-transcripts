### Post #1 — @kushti (2025-03-24)

This update is covering two weeks again, since Mar, 9th! Maybe we’ll switch to bi-weekly updates as everyone is too busy at the moment for weekly ones.

* Nautilus 1.1.0 (and before, 1.0.0) released, with reworked wallet UI and SigUSD bank contract UI! <https://chromewebstore.google.com/detail/nautilus-wallet/gjlmehlldlphhljhpnlddaodbjjcchai?pli=1>
* Satergo 1.9.1 (and before, 1.9.0) released! with Ledger support, offline vault option, update functionality (instead of just opening the website), and more secure wallet format! <https://satergo.com>
* Public testnet based Dexy UI is available (use Nautilus for Testnet to test it) <https://test.ergfi.xyz/>
* Ergo protocol reference client 6.0.0-RC3 released! [Release Ergo protocol reference client 6.0.0-RC3 · ergoplatform/ergo · GitHub](https://github.com/ergoplatform/ergo/releases/tag/v6.0.0-RC3) , now available at Docker Hub and can be easily built locally
* ITurkMining pool has added Ergo support! [iTurkMining.com](https://iturkmining.com/#ergo1)
* The first Duckpools v2 pool is available for testing at <https://test.duckpools.io>
* Mew Mart got updated stats <https://mart.mewfinance.com/> . Also check nice Ergo cups @ Mew Store <https://store.mewfinance.com/>
* Another airdrop tool, now mostly AI generated! <https://x.com/NeuralYogi/status/1902078953818362093>
* As reported by Keystone wallet team, integration is 90% done! <https://x.com/KeystoneWallet/status/1904159866366906524>
* Scrypto 3.1.0 , as well as new versions of debox and scorex-util dependencies, released, with Scala 3 support and more! [Release v3.1.0 · input-output-hk/scrypto · GitHub](https://github.com/input-output-hk/scrypto/releases/tag/v3.1.0)
* Crowdfunding for security audit of 6.0.0 code finished succeed ! [Ergo Raffle](https://ergoraffle.com/raffle/show/850d0c68a883b1d6822148fc20708948e0aa97a768fb976ce3b20fad878ef89a)
* there was good Twitter Spaces event on “Programmable Reserve Assets on Ergo: the Quest for Better Money”, you can find recording at <https://x.com/i/spaces/1jMJgkWDMVyJL/peek> or [Unsupported browser](https://open.spotify.com/episode/2LsOdHjeslLwthApVjNW7Z)
