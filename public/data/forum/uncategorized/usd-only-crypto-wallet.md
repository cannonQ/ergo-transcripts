### Post #1 — @kushti (2021-04-05)

Once I received (ERC-20) USDT into Exodus wallet. However, as wallet was empty before, it wasn’t possible to send them out without getting some ETH to pay fees.

However, in Ergo we can do better, so it possible to have a wallet with just USD shown to users, with mining fees (and min ERG value for a UTXO created) to be paid under the hood via a special contract converting USD to ERG, similarly to “fee emission contract” already used in the ErgoMixer (see [Paying fee in ErgoMix in primary tokens](https://www.ergoforum.org/t/paying-fee-in-ergomix-in-primary-tokens/73) for details).

Then the wallet can have very simple UI, understandable by non-crypto users, and also using crypto features, e.g. stealth addresses and mixer for privacy.
