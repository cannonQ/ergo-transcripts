### Post #1 — @kushti (2024-10-14)

* Another weekly dev update is out <https://www.youtube.com/watch?v=9jlA6eizQIc&t=4s> , so if you prefer video format, this update is for you!
* The countdown has started for the Rosen Bridge launch! From the team: “Over the coming days, we will be implementing our Ethereum bridge. This will occur over a few steps: Administration (Set Configs, issue tokens and update contracts and token maps), Infrastructure (First, Guards and then Watchers to update their services), Test (A couple of private test transfers to check Configs in production), User Interface (Release the UI and the bridge is live to the public!)”
* Ergo protocol reference client 5.0.24 has been released [Release Ergo Protocol Reference Client 5.0.24 · ergoplatform/ergo · GitHub](https://github.com/ergoplatform/ergo/releases/tag/v5.0.24)
* Ergo protocol client 6.0.0-alpha2 for devnets has been released. Contains all the planned Sigma 6.0.0 features [Release Ergo protocol reference client 6.0.0 Alpha2 (for devnets only!) · ergoplatform/ergo · GitHub](https://github.com/ergoplatform/ergo/releases/tag/v6.0.0-alpha2)
* Lithos started testing NISP contracts which has gone well so far.
* Paideia started final on chain testing (but you can use beta on Ergo mainnet already)
* Support for Flex and Stax reportedly done, new code will be integrated into current codebase and sent to Ledger for another review next week
