欢迎阅读Ergo平台5月的月度总结！ 我们很高兴与大家分享我们在这个月取得的令人激动的进展，重点介绍增强功能、项目开发和社区互动。

[**里程碑** **提醒** **！** #ERGO区块链刚刚达到了一个重要的里程碑——第1,000,000个区块的挖掘！这证明了ERGO社区的力量和适应性。](https://twitter.com/ManaStaking/status/1655910294055530496)

2023年7月，平台将开展Ergo四周年庆。 这是一个重要的时刻，值得庆祝的事情太多了！现在了解更多！

请在你的日历做好标记。**2023年7月1日至3日** ，将为Ergo举办一场盛大活动，包括Ergo周年庆和Ergo峰会！无论您是Ergo新人还是已经在该网络上活跃了一段时间，每个人都能找到适合自己的东西。请分享这一消息，Ergonaut，并祝贺我们所有人一起达到这一里程碑。

现在，让我们深入了解更新！

**生态系统更新**

向[Ergoplatform.org](https://ergoplatform.org/)新增了最新成员：实时聊天机器人！这个高级聊天机器人已经过全面的[Ergo文档](https://docs.ergoplatform.org/)的精心培训。试一试，探索其丰富的知识！

整个5月，我们各个敬业的团队一直在不懈努力，以确保Ergo生态的不断发展和完善。

开发者统计数据现在可在[Artemis](https://app.artemis.xyz/developers/Ergo)上获得。 尽管最近出现下滑趋势，但开发者社区继续蓬勃发展，推动了Ergo核心和子生态系统的创新。

**项目** **开发** **：** 我们见证了Ergo生态系统内各种项目的重大进展：

[**Paideia**](https://app.paideia.im/) DAO[现已在链上运行](https://twitter.com/PaideiaDAO/status/1664222742713974784)！ 他们剩下的任务是整合两个组件，为用户提供无缝体验。

**DexyGold** 测试进展顺利，请密切关注，因为在未来几天将共享测试网银行UI的访问

[**Spectrum Finance**](https://spectrum.fi/)

公布了他们已完成所有必需的应用程序接口（API），以便将[Ergo DeFi](https://twitter.com/hashtag/Ergo?src=hashtag_click)对接到Coingecko上！

在成功推出流动性挖矿（Yield Farming）协议后，他们将此功能扩展到生态项目，并允许他们推出自己的流动性矿场（Yield Farm）！

**PalmyraComDex：** [已发布](https://twitter.com/ZengateGlobal/status/1660702433725800449)其最简可行产品（MVP）的路线图。**ZengateGlobal** 还[宣布](https://twitter.com/ErgoPadOfficial/status/1659239229610762240)他们将通过ErgoPad正式推出他们的IDO，使其成为Ergo和Cardano区块链上的首个跨链IDO。

[**Duckpools**](https://www.duckpools.io/)即将全面启动，大多数合约及其链下组件都处于开发的最后阶段。 新的资金池合约已经过彻底的安全审查。

[**罗森跨链桥（** **Rosen Bridge** **）**](https://soft.rosen.tech/)**介绍** 了该项目的汇总并等待代码审查。此外，他们**已开始致力于将基于** **以太坊虚拟机（** **EVM** **）** **的链** **整合** **到Rosen Bridge中的研发工作。**

[**SkyHarbor**](https://skyharbor.io/)完成了Ergo Pay测试，目前正在开发一个自动收藏品创建系统。

[**SigmaUSD**](https://sigmausd.io/)推出了一系列更新。 首先，实现了改进的拥塞处理，从而显着减少了转换出错。其次，SigmaUSD现在直接与银行交互，绕过代理合约。此增强功能可减少出错并消除等待退款的需要。最后，引入了可调整的矿工费，使用户能够有效地与其他人和机器人竞争。

[**Bitdomains**](https://github.com/bitdomains)是Ergo的新名称服务，它克服了前端挑战，现在专注于开发低级和高级AVL树代码库并继续合约开发。

**SigmaFi**最终确定了贷款邀约（*Loan Offers* *）* 这项新功能的合约，该功能现已准备好进行Beta测试。

[**Analog Ergo**](https://docs.ergoplatform.com/eco/analog-ergo/)已完成自动化Ergo脚本合约部署，引入了接收方合约价值验证。

任何拥有最新版**Nautilus** 钱包的人现在都可以使用[single-tx-swap.com](https://www.single-tx-swap.com/)。

**TabbyPOS** [开发更新](https://tabbylab.medium.com/)包括解决普通的编码任务和优化界面，并计划在下个月中旬完成商户门户。此外，该平台明确表示其立场不是跨链，而是在现有的ERGO基础设施之上增加对其他加密货币的支持。其他消息，他们成功获得了 BinancePay的商户账户，AnetaBTC已经在马来西亚建立了业务，很有可能获得加密货币交易许可证，这将极大地促进法币交易。

[**E** **rgopad**](https://ergopad.io/)宣布了[Coinecta Finance](http://coinecta.fi/)，这是一个Cardano代币启动平台，允许他们举办 Cardano IDO 以及跨链Ergo-Cardano IDO。

**钱包：**

[**Nautilus**](https://github.com/capt-nemo429/nautilus-wallet/releases/tag/v0.8.0)现在有一个方便的“dApps”部分，其中包含一个工具，可以让您轻松识别和整合您的UTxO盒子！

[**SAFEW**](https://github.com/ThierryM1212/SAFEW/releases/tag/0.7.2) v0.7.2 带来了一个改进的dApp连接器，包含立即节点地址更新、一个带有版本和高度详细信息的增强节点表，以及每个钱包和地址的盒子数量和存储租赁费时间的改进显示。

**Ergo Wallet App** 专注于增强交易能力和改善用户体验。 这些更新包括错误修复、将交易导出为CSV的功能，以及在选择节点Explorer API时从节点获取交易数据。

**Koinly** 正在推进整合，这是一种加密货币税务软件。这将帮助用户管理他们的投资组合和纳税义务。希望在几周内准备好！

**社区MVP** ：其中包括新浏览器！它通过用户友好的用户界面[ergexplorer.com](https://www.ergexplorer.com/)近乎即时地加载钱包地址

**NFT、游戏和** **代币**

Ergo上的NFT生态系统继续蓬勃发展并吸引着爱好者。

[Ergo Auction House](https://ergoauctions.org/)，是一家拍卖 NFT 和其他物品的去中心化平台，已推出了其平台的第三次迭代！ 此外，您还可以前往Ergopad上质押其代币AHT并赚取收入分成！30%的利润与代币持有者分享，每4周（最初）手动分配一次。

**Comet** 为他们的*Comet Lottery* 推出了一个新的前端，现已与Ergo Pay对接。

Mick，又名LowKeyNerd，[回归Ergo Pulse](https://www.youtube.com/watch?v=yOdaZvLIyno)为我们带来了另一个很棒的更新，并展示了他在过去12个月里在**Blitz TCG** 上取得的惊人进步。Blitz是一款集换式卡牌游戏，它将使用区块链实现游戏资产的真正数字所有权。qx()，也称为 qx 括号，在这一期节目中代表Armeanio出席本期节目，“来自日本”的Dan Friedman也出席了本次节目。

[**Grand Gambit**](http://grandgambit.io/)现在使用户能够直接从他们的钱包中参与游戏，并列出了Paideia和 $BASS代币，允许其用户使用原生代币玩Plinko、Roulette、Coinflip等。

**开发与研究**

Ergo的开发者和研究人员团队不断创新并推动平台的功能。

**参考客户端：** [Node 5.0.11](https://github.com/ergoplatform/ergo/releases/tag/v5.0.11)发布，改进了钱包扫描性能，自动版本替换，更清晰的 API 错误消息，以及其他小修复。5.0.12版已经[在测试中](https://github.com/ergoplatform/ergo/pull/1999)，引入了UTXO集快照，这将大大缩短同步时间！（在普通笔记本电脑上为30-60m）

**sigmastate-** **interpreter** 版本5.0.8和Sigma.JS几乎就在这里。Sigma JS和[Fleet](https://docs.ergoplatform.com/dev/stack/fleet/)的结合将为Javascript开发者提供类似于[AppKit](https://docs.ergoplatform.com/dev/stack/appkit/)为JVM开发者提供的可用性。

[**ergo-lib-go**](https://github.com/sigmaspace-io/ergo-lib-go)**：** sigma-rust c-bindings的go-lang包装器，允许开发者在golang中使用 sigma-rust

Nautilus和Fleet软件开发工具包（SDK）进行了重大升级，以促进不同dApp实现之间的代码重用和一致性。

与Djed联盟讨论新的稳定币设计“**SigGold** ”

kushti一直在跟anon\_real一起探索开发一种基于拍卖的释放和堕落金融自主机器（degen finance autonomous machine），称为 Auction Coin

**社区亮点**

我们的Reddit社区“r/ergonauts”进行了多次引人入胜的讨论、更新和公告。开发者更新、存储租赁费讨论和社区主题是突出的话题。此外，在最近的Ledger事件之后，有人询问了Ergo的Ledger支持。（他们后来推翻了这个决定）

**Ergonauts谈论生活** ：我们的社区不仅仅是区块链和加密货币。在“Ergonauts谈论生活”话题中，我们社区的成员一直在分享他们对生活各个方面的想法和经验。看到我们的Ergonaut人性化的一面总是很欣慰！

**Twitter和Reddit策略** ：我们一直在集思广益，探讨如何有效地使用Twitter和Reddit 等社交媒体平台来传播有关Ergo的信息。这些讨论很有见地，我们很高兴实施这些策略来发展我们的社区。确保加入[Telegram](https://t.me/ErgoSocials)或[Discord](https://discord.gg/ergo-platform-668903786361651200)上的社区营销聊天群。

[**带着** **信任长期投资**](https://www.ergoforum.org/t/lit-long-term-investment-of-trust/4358/3)（LIT）是一个在Ergo生态系统中一直在积极讨论和发展的概念。虽然 LIT的细节仍在完善中，但其核心思想围绕着创建一个系统，该系统鼓励和奖励Ergo网络内的长期投资和信任。

[存储租赁费激活（Storage Rent Activation）](https://www.reddit.com/r/ergonauts/comments/13awj9n/storage_rent_activation_begins_at_block_number/)关于在区块1051200 激活存储租赁费的公告。另请务必查看此信息系列：[*跟踪存储* *租赁费* *潜力|第三版*](https://www.reddit.com/r/ergonauts/comments/13e8f8g/tracking_storage_rent_potential_3rd_ed/)。
