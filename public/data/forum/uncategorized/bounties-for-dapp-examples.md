### Post #1 — @kushti (2021-11-11)

Bounties for submissions to ErgoScript-by-Example repository were very successful, a lot of submissions were received.

There’s also demand for ready-made application example to fully grasp possible full-stack solutions for dApps on Ergo.

Thus there are new bounties! First five submitted and approved applications will receive:

* 2,000 SigUSD for an application with Web interface
* 1,000 SigUSD for an application with command-line interface

All the parts of an application must be fully open-sourced, please choose least restrictive license option possible.

To submit, whether reach #development channel in Discord [Ergo Platform](https://discord.gg/kj7s7nb) and announce there , or drop an email at [team@ergoplatform.org](mailto:team@ergoplatform.org)
