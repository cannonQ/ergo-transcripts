# 罗森跨链桥将成为Ergo、比特币等不同区块链的交汇点

随着罗森跨链桥（Rosen Bridge）的启动，去中心化的区块链基础设施得到了显着的推动。 截至目前，Ergo和Cardano生态系统的参与者可以在这两条不同区块链之间交换原生资产，且该跨链桥将很快扩展到比特币、以太坊、币安智能链和狗狗币等区块链，实现这些区块链上的原生资产交换。

随着罗森跨链桥的出现，为Ergo和Cardano区块链的代币持有者带来了新的机会。现在，Ergo原生代币在Cardano上，反之亦然，Cardano代币在Ergo上，因此有新的流动性池（LP）可用。

流动性池是资金存储器（这些资金存放在智能合约中），存放有构成交易对的两种资产。例如，如果有人在自己的钱包中持有ADA并想要获取rsERG（Cardano上的ERG代币），他们将访问Cardano DEX（去中心化交易所）上的ADA/rsERG流动性池。通过该流动性池，他们将能够将ADA交换为rsERG（同样，他们可以在同一流动性池中将rsERG交换为ADA）。

去中心化金融的目标之一是允许任何人在没有任何中介的情况下充当银行实体的角色。他们可以是贷款人或者流动性提供者。因此，在去中心化交易所（DEX）上持有交易对代币的用户可以使用这些资产提供流动性并赚取奖励（他们在流动性池中的投资回报）。两种资产之间的交易活跃度是否健康取决于该流动性池的可用流动性金额。更大的流动性池确保有足够的流动性来支持各自资产之间的大量交易。

随着Cardano和Ergo的新原生代币在两个区块链之间流动，让我们来看看投资者可以使用的一些新流动性池。

**加密货币投资者的机会**

在罗森跨链桥启动的早期阶段，不仅Spectrum Finance 和Minswap的这些流动性池具有投资潜力，而且流动性提供者还有机会为其他原生资产创建新的流动性池，例如 SigUSD、SigRSV、HOSKY 和 COMET 。 例如，新投资者可能会投资其代币来创建 SigUSD/rsADA（Ergo端）或 ADA/rsSigUSD（Cardano端）等交易对。

Ergo由加密货币背书的创新型算法稳定币（[SigUSD](https://sigmausd.io/#/)）将是Cardano区块链的一个非常有趣的补充。加密行业内的许多人认为，区块链的成功取决于成功部署的稳定币协议的使用。通过 SigUSD，投资者可以获得一种稳定币，这种稳定币不仅是去中心化的，而且在 ERG 价格极端波动期间也能保持与美元挂钩。

罗森跨链桥开发者目前正在努力向该跨链桥添加更多原生资产，[以太坊似乎将成为下一个添加到](https://x.com/RosenBridge_erg/status/1747861451375833286?s=20)罗森跨链桥Watcher和Guard系统中的区块链。所有三个生态系统的原生资产都可以在各自的区块链DEX上进行交易只是时间问题。

**桥接不同区块链：提供去中心化基础设施**

与比特币或Ergo的工作量证明共识保护网络并批准交易的方式大致相同，[罗森跨链桥的设计使任何人都可以成为Watcher并帮助保护该跨链桥的安全](https://ergoplatform.org/en/news/Rosen-Bridge-Officially-Launches-on-Ergo-Mainnet/)。作为提供这一重要基础设施的交换，Watcher将获得跨链桥交易手续费（以RSN代币支付）作为奖励。罗森跨链桥网络中的每个区块链都需要Watcher来观察交易。随着该跨链桥扩展到更多区块链，人们将需要运行 Watcher 软件并帮助保护跨链桥交易。 如果您不是编码员，这可能看起来是一项复杂的任务，但实际上使用 [Rosen Bridge Github](https://github.com/rosen-bridge) 部署Watcher非常容易。

**设置** **Watcher**

如果您有兴趣运营Watcher并赚取RSN被动收入，您将需要访问以下链接：

[Ergo Rosen Watcher 设置 (Windows) Youtube 视频](https://www.youtube.com/watch?v=bQ2sviHtOQA)

[适用于 Windows 的 Git（适用于Windows Setup的 64 位 Git）](https://git-scm.com/download/win)

[适用于 Windows 的 Docker 桌面](https://docs.docker.com/desktop/install/windows-install/)

[Github上的 Rosen Bridge Watcher须知](https://github.com/rosen-bridge/operation/blob/dev/docs/watcher/deploy-docker.md)

在设置 Watcher 之前，您需要安装 64 位 Git 和 Docker Desktop。 安装完毕后，您可以借助Ergo Rosen Watcher 设置 Youtube 视频教程和 Github须知来部署您自己的Watcher。 设置视频教程非常有用，可以引导观看者完成从开始到结束的整个过程。作者不是编码员，但按照视频中的分步过程成功设置了Cardano watcher。

请注意，为了运行Watcher，您需要30k RSN 和 800 ERG 的初始投资。这些资金将作为抵押品锁定在跨链桥中。 此外，对于在您的Watcher上运行的每个许可证，您还需要3000 RSN。 您的Watcher拥有的许可越多，您可以见证的跨链桥事件就越多，从而可以瓜分交易手续费。

**区块链互操作性的未来**

作为一种去中心化工具，罗森跨链桥有潜力开启区块链互操作性时代。随着越来越多的区块链被添加到跨链桥基础设施中，开发者将能够以一种全新的、革命性的方式看待 dApp 设计。罗森跨链桥预先存在的跨链基础设施为其dApp打开了大门，以充分利用不同区块链的各种优势。

如果您是投资者、开发者或生态系统参与者，可以通过多种方式与罗森跨链桥互动。新的流动性池现已通过Minswap和Spectrum Finance 在 Cardano 和 Ergo 上提供。 开发者可以探索和研究通过 Ergo 的工作量证明和Cardano的权益证明共识模型为其 dApp 提供支持的方法。 此外，任何拥有计算机的人都可以作为Watcher帮助保护跨链桥的安全并赚取被动收入。

欲了解更多信息并访问该跨链桥，请访问[罗森跨链桥官网](https://rosen.tech/)。 如果您有任何其他问题或需要设置Watcher的帮助，感兴趣的各方可以访问 [Rosen Bridge Telegram](https://t.me/rosenbridge_erg) 和/或 [Discord](https://discord.com/invite/AHgsxhDKrQ)。 在那里，您可以找到知识渊博的社区，随时可以帮助解决任何问题。
