# Change log

### Nanominer v3.2.0 / release 05.02.21

* Added Autolykos algorithm support for Ergo coin. Fee is 5%. Needs at least 3 GB GPU. Hashrates on stock settings:
* Nvidia 3070: 143 MH/s
* Nvidia 3060 ti: 142 MH/s
* Nvidia 1060: 33 MH/s
* AMD RX 5700: 81 MH/s
* AMD RX Vega 64: 80 MH/s
* AMD RX 580: 42 MH/s

### [Nanominer - best cryptocurrency miner!](https://nanominer.org)

Nanominer is a versatile tool for mining cryptocurrencies which are based on Ethash,Ubqhash CryptoNight (v6, v7, v8) and RandomHash (PascalCoin) algorithms.
