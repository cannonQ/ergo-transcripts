## Ergo Platform Ecosystem Digest - March 2020

* Anon92048 open-sourced command-line mixing tool with instructions <https://github.com/anon92048/ergo-mixer-demo> Introducing Σ3 - ErgoMixer, a powerful mixer that works by pooling funds and enabling participants to spend them without anyone knowing who made the transactions <https://ergoplatform.org/en/blog/2020_03_20_ergo_mixer/>
* DEX transactions are done in the mainnet, DEX demo is coming with instructions and the command-line tools, then UI for DEX and mixing will become the main issue
* Ergo Platform Reference Client 3.2.2 released <https://github.com/ergoplatform/ergo/releases/tag/v3.2.2>
* jasondavies got Ergo Foundation grant to implement a hardware wallet, anon92048 works on automating mixing, two exchange listings are coming
* filippounits successfully tested light node on Ras Pi with 1 GB RAM, now testing full node <https://twitter.com/FilippoUnits/status/1243610791577899008>
* ErgoTree Specification for Ergo Protocol 1.0 released <https://ergoplatform.org/docs/ErgoTree.pdf>
* ErgoTree & ErgoScript interpreter 3.2.1 released <https://github.com/ScorexFoundation/sigmastate-interpreter/releases/tag/v3.2.1> Certified Ergo contracts repository released <https://github.com/ergoplatform/ergo-contracts>
* BigEvilSoloMiner announced that ERG mining will be suspended indefinitely on <http://erg.ezpool.io/> beginning tonight after payouts
